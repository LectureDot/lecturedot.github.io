LectureDocs

LectureDocs is a collection of higher education assessment tools.
  • LectureSeats - mark lecture attendance using QR code with optional selfies, seat code, and geolocation stamps
  • LectureTest - secure exam-taking with multiple questions types (including 2-d Parsons and coding problems) and automatic grading
  • LectureAssign - Visual Studio Code extension for streamlining coding assignments and reducing plagiarism
